# 毕业设计文档

## 项目名称
智能校园活动管理系统

## 技术架构

### 前端技术栈
- **框架**: Vue 3.4.15 (Composition API)
- **构建工具**: Vite 5.0.11
- **状态管理**: Pinia 2.1.7
- **路由管理**: Vue Router 4.2.5
- **UI 组件库**: Element Plus 2.5.4
- **HTTP 客户端**: Axios 1.6.5
- **AI 集成**: DeepSeek API

### 核心特性
1. **前后端分离架构**
2. **组合式 API 开发模式**
3. **AI 智能功能集成**
4. **响应式设计**
5. **权限管理系统**

### 3. 核心功能实现

#### 3.1 状态管理 (Pinia)

**用户状态管理 (user.js)**
```javascript
- 用户信息存储
- 登录/注册/登出
- 收藏/报名管理
- 浏览历史记录

```

**活动状态管理 (activity.js)**
```javascript
- 活动列表管理
```

**AI 状态管理 (ai.js)**
```javascript
- AI 推荐结果缓存

- 评论分析结果
- 数据洞察报告
```

#### 3.2 路由管理 (Vue Router)

**路由配置**
- 学生端路由：首页、活动列表、活动详情、搜索、个人中心
- 管理员路由：数据概览、活动管理、AI 分析
- 路由守卫：权限验证、登录检查

**权限控制**
```javascript
- requiresAuth: 需要登录
- requiresAdmin: 需要管理员权限
- 自动重定向到登录页
```

#### 3.3 AI 功能集成

**DeepSeek API 集成**
```javascript
// 服务封装
class DeepSeekService {
  - chat()                    // 基础对话
  - recommendActivities()     // 智能推荐
  - enhanceSearch()           // 搜索增强
  - extractSummary()          // 摘要提取
  - analyzeComment()          // 评论分析
  - analyzeData()             // 数据分析
  - generateTitleSuggestions() // 标题生成
}
```

**AI 功能实现原理**

1. **智能推荐**
   - 输入：用户行为数据（收藏、报名、浏览历史）
   - 处理：DeepSeek 分析用户兴趣偏好
   - 输出：推荐活动 ID 列表

2. **智能搜索**
   - 输入：搜索关键词 + 活动列表
   - 处理：理解搜索意图，提取相关信息
   - 输出：相关活动、同义词、筛选建议

3. **评论分析**
   - 输入：评论文本
   - 处理：情感分析、内容审核
   - 输出：情感倾向、规范性建议

4. **内容摘要**
   - 输入：活动详细描述
   - 处理：提取关键信息
   - 输出：摘要、关键词、亮点

5. **数据分析**
   - 输入：平台运营数据
   - 处理：趋势分析、规律发现
   - 输出：洞察、趋势、建议

#### 5.3 Vue Router 路由管理

**路由守卫**
```javascript
router.beforeEach((to, from, next) => {
  // 权限检查
  if (to.meta.requiresAuth && !isLoggedIn) {
    next('/login')
  } else {
    next()
  }
})
```

#### 5.4 Axios 请求封装

**拦截器**
```javascript
// 请求拦截器：添加 token
request.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})

// 响应拦截器：统一错误处理
request.interceptors.response.use(
  response => response.data,
  error => {
    // 错误处理
    return Promise.reject(error)
  }
)
```
