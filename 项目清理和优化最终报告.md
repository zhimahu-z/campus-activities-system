## ⚠️ 注意事项

### 1. 异步方法调用

所有涉及报名和收藏的方法现在都是异步的，需要使用 `await`：

```javascript
// 错误 ❌
userStore.registerActivity(activityId)

// 正确 ✅
await userStore.registerActivity(activityId)
```

### 2. 错误处理

所有API调用都有错误处理，返回格式统一：

```javascript
const result = await userStore.registerActivity(activityId)
if (result.success) {
  // 成功
} else {
  // 失败，result.message 包含错误信息
}
```

### 3. 数据同步

- 报名/取消报名后会触发 `activity-registration-changed` 事件
- 相关组件会自动刷新数据
- 无需手动刷新页面
