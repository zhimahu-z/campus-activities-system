## 🔧 功能详细说明

### 一、取消报名功能

#### 功能位置

**1. 活动详情页（ActivityDetail.vue）**
- 已报名的活动，"立即报名"按钮会变成"取消报名"按钮
- 按钮颜色：红色（danger）
- 点击后会弹出确认对话框

**2. 个人中心 - 我的报名（My.vue）**
- 每个已报名的活动卡片下方有"取消报名"按钮
- 按钮颜色：红色（danger）
- 点击后会弹出确认对话框

#### 使用流程

```
1. 学生登录系统
   ↓
2. 进入活动详情页 或 个人中心
   ↓
3. 点击"取消报名"按钮
   ↓
4. 弹出确认对话框："确定要取消报名吗？"
   ↓
5. 点击"确定" → 取消成功，显示提示"已取消报名"
   点击"取消" → 不执行任何操作
```

#### 技术实现

**Store方法**（`src/stores/user.js`）：
```javascript
const unregisterActivity = (activityId) => {
  const index = registrations.value.indexOf(activityId)
```

- 已收藏：实心星星（StarFilled）

#### 使用流程

```
1. 学生登录系统
   ↓
2. 进入活动详情页 或 个人中心
   ↓
3. 点击"取消收藏"按钮
   ↓
4. 弹出确认对话框："确定要取消收藏吗？"
   ↓
5. 点击"确定" → 取消成功，显示提示"已取消收藏"
   点击"取消" → 不执行任何操作
```

#### 技术实现

**Store方法**（`src/stores/user.js`）：
```javascript
const toggleFavorite = (activityId) => {
  const index = favorites.value.indexOf(activityId)
  if (index > -1) {
    favorites.value.splice(index, 1) // 取消收藏
  } else {
    favorites.value.push(activityId) // 添加收藏
  }
  // 保存到 localStorage
  if (userInfo.value) {
    const userId = userInfo.value.id
    localStorage.setItem(`favorites_${userId}`, JSON.stringify(favorites.value))
  }
}
```

**组件调用**：
```javascript
const handleUnfavorite = (activityId) => {
  ElMessageBox.confirm('确定要取消收藏吗？', '提示', {
    confirmButtonText: '确定',
    cancelButtonText: '取消',
    type: 'warning'
  }).then(() => {
    userStore.toggleFavorite(activityId)
    ElMessage.success('已取消收藏')
  }).catch(() => {
    // 用户取消操作
  })
}
```

### 三、活动状态标识

#### 状态类型

| 状态 | 显示文本 | 颜色 | 判断条件 |
|------|----------|------|----------|
| **未开始** | 未开始 | 紫色渐变 | 当前时间 < 开始时间 |
| **进行中** | 进行中 | 粉色渐变 + 脉动动画 | 开始时间 ≤ 当前时间 ≤ 结束时间 |
| **已结束** | 已结束 | 灰色 | 当前时间 > 结束时间 |

#### 显示位置

**所有页面都会显示活动状态标识：**

1. ✅ **首页（Home.vue）**
   - 位置：活动卡片封面图右上角
   - 样式：圆角徽章，半透明背景

2. ✅ **活动列表（Activities.vue）**
   - 位置：活动封面图右上角
   - 样式：圆角徽章，半透明背景

3. ✅ **智能搜索（Search.vue）**
   - 位置：活动卡片封面图右上角
   - 样式：圆角徽章，半透明背景
   - 同时显示"AI推荐"标识（左上角）

4. ✅ **活动详情页（ActivityDetail.vue）**
   - 位置：右侧操作卡片顶部
   - 样式：大号状态徽章，带图标

5. ✅ **个人中心（My.vue）**
   - 位置：活动卡片封面图右上角
   - 样式：圆角徽章，半透明背景
   - 显示在：我的报名、我的收藏、浏览历史

#### 视觉效果

**未开始（紫色渐变）**
```css
background: linear-gradient(135deg, rgba(102, 126, 234, 0.9), rgba(118, 75, 162, 0.9));
color: white;
```

**进行中（粉色渐变 + 脉动动画）**
```css
background: linear-gradient(135deg, rgba(240, 147, 251, 0.9), rgba(245, 87, 108, 0.9));
color: white;
animation: pulse 2s ease-in-out infinite;
```

**已结束（灰色）**
```css
background: rgba(224, 224, 224, 0.9);
color: #666;
```

#### 技术实现

**状态判断函数**：
```javascript
const getActivityStatus = (activity) => {
  const now = new Date()
  const startTime = new Date(activity.startTime)
  const endTime = new Date(activity.endTime)
  
  if (now < startTime) return 'upcoming' // 未开始
  if (now >= startTime && now <= endTime) return 'ongoing' // 进行中
  return 'ended' // 已结束
}

const getActivityStatusText = (activity) => {
  const status = getActivityStatus(activity)
  const statusMap = {
    upcoming: '未开始',
    ongoing: '进行中',
    ended: '已结束'
  }
  return statusMap[status] || '未知'
}
```

**模板使用**：
```vue
<div class="activity-status" :class="`status-${getActivityStatus(activity)}`">
  {{ getActivityStatusText(activity) }}
</div>
```
